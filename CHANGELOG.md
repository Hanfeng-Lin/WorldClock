- Added a time converter dialog for converting a selected city/date/time to current widget cities or all built-in cities.

- Added city search and ordering in the city picker.
- Added lock position and reset position actions.
- Added click-through mode.
- Added seconds display toggle.
- Added font size presets.
- Added width presets.
- Improved Explorer restart repinning.
- Improved multi-monitor position recovery.
- Added per-monitor DPI manifest settings.
- Reduced desktop pinning log noise.
- Added `.gitignore`, `TODO.md`, `LICENSE`, and changelog.

- Added opacity menu.
- Added smooth per-pixel alpha rounded corners.
- Added bilingual UI.
- Added theme switching.
- Added 12/24-hour time format switching.
